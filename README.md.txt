# 🎮 Tic-Tac-Toe AI Web App

An interactive and fun Tic-Tac-Toe game built with **Streamlit**, featuring:

- 🧠 Unbeatable AI using the **Minimax algorithm with Alpha-Beta Pruning**
- 🤝 Multiplayer and Single Player modes
- 🎭 Emoji-enhanced themed board
- 🕹️ Easy, Medium, Hard difficulty levels
- 🔥 Win streak and score tracking
- ⏱️ Per-move time limit
- 🎧 Sound effects for win/loss/draw
- 📜 Game history with downloadable logs
